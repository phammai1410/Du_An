# 3. MÔ TẢ HỌC PHẦN (Course Descriptions)

Học phần này giới thiệu các căn bản lập trình với Python như biến, các dạng dữ liệu danh sách, cú pháp điều khiển, thiết kế chương trình với hàm và module, lập trình hướng đối tượng với ngôn ngữ lập trình Python. Học phần này cũng giới thiệu phương pháp tải dữ liệu, phân tích dữ liệu, phương pháp thăm dò và trực quan hóa dữ liệu với Python; làm việc với các thư viện phổ biến của Python như: Pandas, Numpy, Matplotlib.

Học phần cũng nhằm rèn luyện kỹ năng tư duy, khả năng tự tìm hiểu các kiến thức về ngôn ngữ lập trình Python bằng cách làm việc độc lập cũng như theo nhóm. Sinh viên sẽ được yêu cầu xây dựng một ứng dụng thực tế bằng Python.

**Nội dung chính:**

* Giới thiệu về Python
* Các kiểu dữ liệu trong Python
* Các cấu trúc lập trình
* Hàm, module, packages trong Python
* Làm việc với tệp trong Python
* Lập trình hướng đối tượng với Python
* Phân tích và xử lý dữ liệu với Python

## 10.4. Quy định về nộp bài tập lớn

* Người học không nộp bài cá nhân / nhóm theo yêu cầu sẽ nhận điểm 0 (không).
