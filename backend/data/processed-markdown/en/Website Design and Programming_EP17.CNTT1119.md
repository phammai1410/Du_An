# 2. COURSE DESCRIPTIONS

This course introduces the concept of websites, various web architecture models, and methods for designing, building, and deploying a website. Additionally, it guides students through the complete step-by-step process of developing an e-commerce website with commonly used features using one of the popular web programming languages such as C#, Java, JavaScript, or PHP.

The content includes the following:

* Basic of Website
  + Concepts of Web architecture and Web services
  + Designing, Developing, and Deploying a website
  + Steps for building an e-commerce website.
* Design a website.
  + Overview of Website design
  + Principles of HTML
  + CSS
* Building a website
  + Fundamentals of Programming Languages
  + Basic Syntax
  + Working with method
  + Working with Files
  + Principles of Object-Oriented Programming (OOP)
  + Working with basic web controls
  + Working with Validation controls
  + Working with Sessions
  + Working with Data controls
  + Working with Databases
  + Working with MVC Model
  + Working with Web API

# 8. COURSE REQUIREMENT & EXPECTATION

## 8.1. Requirements for taking final /terminal exams.

* Students are allowed to take the final exam if their attendance score reaches 5 points or more (on the scale of 10).

## 8.2. Requirements for attending classes.

* Students are responsible for attending all classes. In the case of absence from school due to force majeure reasons, sufficient and reasonable proofs must be provided. For each absence, 1 point will be deducted 1 point. Students who miss any classes more than 3 times, with or without reason, will be considered as failing to complete the course and have to re-register.
* Students will be awarded points for constructive comments. These points are added to attendance points and group assignments.
* Groups who do not submit the group work will receive a score of 0 (zero). Late submissions will be deducted for each day of late submission.

## 8.3. Requirements for in-class behavior

* The course is conducted on the principle of respecting learners and lecturers. Any behavior that affects the teaching and learning process is strictly prohibited.
* Students must come to class on time. Students who are late more than 10 minutes after class starts will not be allowed to attend the class. Do not make noise and affect others during the learning process.
* Laptops and tablets are only used for the purpose of taking notes for and calculating for lectures and exercises, absolutely forbidden to use for other purposes.
