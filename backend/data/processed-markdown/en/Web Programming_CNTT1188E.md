# 3. COURSE DESCRIPTIONS

This course provides students with Web Programming concepts such as Web Application architecture; How to design, build and deploy a website. This course focuses on server-side technologies. So, learners are expected to have prior working knowledge of HTML, CSS, and JavaScript. You will learn how to implement databases, work within Server-Side environments such Java, NodeJS, PHP, .NET, Python etc., and communicate to the client side through a RESTful API.

# 9. COURSE REQUIREMENTS AND EXPECTATION

## 9.1. Attendance Requirements

* Students are required to fully take part in class lectures and activities.
* Students are allowed to be absent for a maximum of 30% of the class hours. Students who are absent over 30% are not qualified to take the final exam.

## 9.2. In-class Behavior Requirements

* The course is carried out with respect to both lecturer and students. All behavior that might adversely affect the stakeholders is prohibited.
* Students need to be in class on time. Students being late for more than 10 minutes will not be eligible to take part in the session.
* Cellphones, laptops, tablets are only accepted for taking notes of the lectures, preparing course assignments, not for another entertainment purpose while in class duration.
